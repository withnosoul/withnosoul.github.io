## 宏任务和微任务
涉及关键词：Task Queue、Event Loop、Macro Task、Micro Task。

1. 宏任务 
   1. `setTimeout`、`setinterval`、`setImmediate` 
   2. `I / O`
   3. UI Rendering 等 
2. 微任务 
   1. `process.nextTick`
   2. `Promise` 
   3. `Object.observer` （已废弃）
   4. `MutationObserver` （webAPIs）等
