## ES版本说明
和ECMAScript有关的标准有ECMA262，ECMA290，ECMA327，ECMA357，ECMA402，ECMA404，ECMA414等等。

其中：
- 290，327，357等等没有推广开来，被废弃。 
- ECMA 262 是语言规范本身。 
- ECMA 402 则是制定一些基于ECMAScript5或者之后版本的一些国际化API标准。 
- ECMA 404 是JSON规范。 
- ECMA 414 则规定了哪些规范是和ECMAScript有关的。目前内部就包含了262，402和404.
