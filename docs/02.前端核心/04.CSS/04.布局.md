## 布局方式
[MDN CSS Layout](https://developer.mozilla.org/zh-CN/docs/Learn/CSS/CSS_layout)

1. 静态布局 
2. 流式布局（百分比布局） 
3. 响应式布局（rem/em布局） 
4. Flex Layout 
5. Grid Layout

::: tip 说明
- 元素的外部显示类型将决定该元素在流式布局中的表现，块级或内联元素。 
- 元素的内部显示类型可以控制其子元素的布局，流式布局、grid 布局或 flex 布局。
:::
