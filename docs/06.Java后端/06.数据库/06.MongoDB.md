## 概念
| SQL术语	| MongoDB术语	| 解释 |
| :---- | :---- | :---- |
| database	| database	| 数据库 | 
| table	| collection	| 数据库表/集合 | 
| row	| document	| 数据记录行/文档（一个对象）|
| column	| field	| 数据字段/域（对象属性名）|
| index	| index	| 索引 |
| table | joins	| 表连接，MongolianDB不支持 |
| primary | key	primary key	| 主键，MongoDB自动将_id字段设置为主键 |
