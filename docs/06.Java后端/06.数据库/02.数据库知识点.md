## 数据库连接池
对于一个简单的数据库引用，用于对数据库的访问不是很频繁，这种情况可以简单的在需要访问数据库时， 
创建一个链接，用完关闭它，这样做不会有太明显的性能上的开销。但是对于复杂的数据库引用，
情况就截然不同了，频繁的建立、关闭连接，会极大的减低系统的性能，这是对于连接的使用造成系统性能的瓶颈。

## 数据源
数据库连接池也叫数据源，英文名叫 DataSource
